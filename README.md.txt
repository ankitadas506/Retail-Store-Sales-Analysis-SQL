# Retail Store Sales Analysis using SQL

## Project Overview

This project analyzes retail store sales data using SQL. A relational database was designed with 5 tables:

* Customers
* Categories
* Products
* Orders
* Order Items

## SQL Concepts Used

* SELECT
* WHERE
* GROUP BY
* HAVING
* INNER JOIN
* LEFT JOIN
* Subqueries
* CASE WHEN
* Date Functions
* Window Functions (RANK, PARTITION BY)

## Business Insights

* Monthly Revenue Analysis
* Revenue by Product Category
* Top Customers by Spending
* Customer Ranking
* Products Ranked Within Categories
* Revenue Contribution Percentage

## Tools Used

* MySQL
* MySQL Workbench
* GitHub
